# BFS-Implementation
CSCI 223 Project

To run the program, hit run and the terminal will prompt you to enter a selection of given characters to make a 2-Dimensional grid which will be used for the BFS algorithm to find the short path from start to goal. Be sure to clearly label your start as 'S' and have it be the first character and the last character be the goal 'G'. The runtime complexity of this operation is O(K).
